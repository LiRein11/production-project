## Сущность счетчика

#### Public api

-   Components

`Counter` - Компонент со счётчиком

-   types

`CounterSchema` - Тип, описывающий редюсер

-   selectors

`getCounter` - Селектор для получения счётчика

`getCounterValue` - Селектор для получения значения счётчика

-   slices

`counterSlice` - Слайс для счётчика
